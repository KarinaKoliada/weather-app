# Weather App

A simple Next.js weather app that allows you to search for cities and view current weather information, including temperature, conditions, wind, humidity, and more.

üåç **Live demo:** [https://weather-b6ke6msm4-karinas-projects-0ca67de4.vercel.app/weather](https://weather-b6ke6msm4-karinas-projects-0ca67de4.vercel.app/weather)


Features

- Search for any city to see current weather
- Displays temperature, high/low, weather description, feels-like temperature
- Highlights key weather stats: wind speed, humidity, sunrise, pressure, visibility
- Responsive layout with a glassy card design
- **Future improvements**: dynamic background, night mode with stars, hourly forecast



Tech Stack

- Next.js 13
- React
- Tailwind CSS
- OpenWeatherMap API
- TypeScript 
- Vercel for deployment
